# 🚀 CIVIX – Smart Street Issue Management System

---

## 📌 Project Overview

CIVIX is an intelligent governance platform designed to empower citizens to report public issues and track them in real-time.

The system ensures:
- Transparency in complaint handling  
- Accountability across government levels  
- Automated escalation using SLA monitoring

### 🧭 Escalation Hierarchy (Core Concept)

![Hierarchy](https://raw.githubusercontent.com/VANITHA1011/complaint_management_system/main/images/hierarchylevel.png)

#### 🔍 Analysis

- Defines complete complaint flow system  
- Four-level escalation model:
  - Level 1: Ward Officer  
  - Level 2: Municipality  
  - Level 3: District  
  - Level 4: State Authority  
- SLA-based progression ensures timely resolution  
- Prevents complaints from being ignored  

---

## 🔄 Complaint Flow

Citizen → Ward (L1) → Municipality (L2) → District (L3) → State (L4)
