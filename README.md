# 🚀 CIVIX – Intelligent Complaint Management System

---

## 📌 Project Overview

CIVIX is an intelligent governance platform designed to empower citizens to report public issues and track them in real-time.
The system ensures transparency and accountability through multi-level escalation and SLA-based monitoring.
